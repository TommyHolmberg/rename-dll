# `rename-dll`

> CLI for renaming DLL files on Windows, generating a new .lib file

## Usage

```ps
rename-dll source.dll dest.dll
```

## API

### `rename-dll ./path/to/source.dll ./path/to/new.dll`

Generate a `.def` file based on `source.dll`,

## Install

```sh
npm install rename-dll
```
